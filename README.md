## CARA INSTALL
# TERMUX
```bash
> download termux
> buka
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> apt update && apt upgrade
> git clone https://github.com/alfiansx/alfbot
> cd alfbot
> bash install.sh
> npm i node-tesseract-ocr
> pkg install tesseract
> npm i
> node index.js
```

## CARA INSTALL
# LINUX
```bash
> apt-get install wget git curl ffmpeg tesseract
> install nodejs on [Here](https://github.com/nodesource/distributions/blob/master/README.md)
> apt update && apt upgrade
> git clone https://github.com/AthallahDzaki/kelliot
> cd kelliot
> chmod +x install.sh
> bash install.sh
> npm i node-tesseract-ocr
> npm i -g cwebp ytdl 
> npm i got
> npm i
> node index.js
```
